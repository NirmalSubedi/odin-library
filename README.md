# Library
Webpage Library to manage and modify a collection of books.

[Browse the Library!](https://nirmalsubedi.github.io/odin-library/)


## How to use?
Open the link above "Browse the Library!".

### Toggle book read status:
Click the book card. 

Note:
- Icon with the check mark indicates you have read the book.
- Icon without the check mark indicates you have not read the book.

### Add a book:
Click "New Book" button, then enter in the book information: title, author, pages, and whether you have read the book or not.

### Close Book Information Dialog
Press `esc` / Click 'x' button at top right.

### Remove a book:
In the book card, click the 'X' button at top right. 

## Which skills were demonstrated?
- Creation of object through the function constructor.
- Initialize an object with the `new` keyword.
- Invalidate any function call without `new` to prevent accidental object construction.
- Utilizing prototypical inheritance to pass methods to instances of the object.
- Referencing object with `this` keyword to set properties when calling function.
